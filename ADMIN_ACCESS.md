# 管理员访问说明

## 本地开发环境

要访问管理后台，请按照以下步骤操作：

1. 确保后端服务器正在运行：
   ```bash
   cd backend && python3 main.py
   ```

2. 确保前端服务器正在运行：
   ```bash
   npm run dev
   ```

3. 访问应用：
   - 前端地址：[http://localhost:3000](http://localhost:3000)
   - 后端API地址：[http://localhost:8000](http://localhost:8000)

4. 使用以下凭据登录以获得管理员权限：
   - 邮箱：`admin@example.com`
   - 密码：任意密码（在演示模式下）

5. 登录后，侧边栏将显示"管理后台"入口

## 管理后台功能

登录为管理员后，您可以访问以下功能：

- **仪表板**：查看系统统计数据
- **用户管理**：查看和管理用户账户
- **内容管理**：审核和管理用户创建的内容
- **系统设置**：配置系统参数

## 在线部署版本

如果您将应用部署到在线平台，管理员访问方式相同，只需将URL替换为您的部署地址。

例如，如果您的应用部署在 `https://my-knowledge-app.vercel.app`，则：

1. 访问：`https://my-knowledge-app.vercel.app`
2. 使用邮箱 `admin@example.com` 登录
3. 管理后台将在 `/admin` 路径下可用

## 重要提醒

- 在生产环境中，请确保使用安全的认证机制
- 不要在生产环境中使用硬编码的管理员账户
- 定期更新管理员密码
- 监控管理员活动日志
