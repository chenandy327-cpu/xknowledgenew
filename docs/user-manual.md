# x² Knowledge Nebula User Manual

## 1. Introduction

### 1.1 About This Manual
This user manual provides comprehensive instructions for using the x² Knowledge Nebula platform, a knowledge sharing and social networking application. The manual covers all major features and functionalities, including user registration, content creation, course enrollment, social interactions, and administrative tasks.

### 1.2 Getting Started
To access the x² Knowledge Nebula platform, users need a compatible web browser (Chrome, Firefox, Safari, or Edge) and an internet connection. The platform is responsive and works on desktop, tablet, and mobile devices.

### 1.3 System Requirements
- **Web Browser**: Latest version of Chrome, Firefox, Safari, or Edge
- **Internet Connection**: Stable broadband connection recommended
- **Screen Resolution**: 1024x768 or higher
- **JavaScript**: Enabled
- **Cookies**: Enabled for authentication and session management

## 2. User Registration and Login

### 2.1 Creating a New Account
1. **Access the Registration Page**:
   - Navigate to the platform's URL
   - Click on the "Register" or "Sign Up" button

2. **Fill in the Registration Form**:
   - **Name**: Enter your full name
   - **Email**: Enter a valid email address
   - **Password**: Create a strong password (at least 6 characters)

3. **Submit the Form**:
   - Click the "Register" button
   - You will receive a confirmation message
   - You will be automatically logged in to the platform

### 2.2 Logging In
1. **Access the Login Page**:
   - Navigate to the platform's URL
   - Click on the "Login" or "Sign In" button

2. **Enter Your Credentials**:
   - **Email**: Enter the email address you used during registration
   - **Password**: Enter your password

3. **Submit the Form**:
   - Click the "Login" button
   - You will be redirected to the home page if your credentials are correct

### 2.3 Resetting Your Password
1. **Access the Password Reset Page**:
   - On the login page, click on "Forgot Password?"

2. **Enter Your Email**:
   - Provide the email address associated with your account
   - Click "Reset Password"

3. **Follow the Instructions**:
   - Check your email for a password reset link
   - Click the link and follow the instructions to create a new password

## 3. Navigation and User Interface

### 3.1 Main Navigation
The platform's main navigation is accessible through the sidebar, which includes:

- **Local**: Explore local content and connections

### 3.3 User Menu
- **Profile**: Access your personal profile
- **Settings**: Manage account settings

## 4. Content Management

### 4.1 Creating New Content
1. **Access the Content Creator**:
   - Click the "Create" button in the sidebar
   - Select "Create Content" from the dropdown menu

2. **Fill in the Content Form**:
   - **Title**: Enter a descriptive title for your content
   - **Content**: Enter your knowledge content (supports rich text formatting)
   - **Category**: Select an appropriate category for your content
   - **Cover Image**: Upload an optional cover image

3. **Submit Your Content**:
   - Click the "Publish" button
   - Your content will be published and visible to other users

### 4.2 Editing Content
1. **Navigate to Your Content**:
   - Go to your profile page
   - Click on the "Content" tab
   - Find the content you want to edit

2. **Edit the Content**:
   - Click the "Edit" button next to the content
   - Make your changes to the content form
   - Click "Update" to save your changes

### 4.3 Deleting Content
1. **Navigate to Your Content**:
   - Go to your profile page
   - Click on the "Content" tab
   - Find the content you want to delete

2. **Delete the Content**:
   - Click the "Delete" button next to the content
   - Confirm the deletion in the popup dialog
   - The content will be permanently removed

## 5. Course Management

### 5.1 Browsing Courses
1. **Access the Course Catalog**:
   - Click on "Courses" in the sidebar

2. **Filter and Search Courses**:
   - Use the search bar to find specific courses
   - Use filter dropdowns to narrow by category, level, or duration

3. **View Course Details**:
   - Click on a course card to view detailed information
   - Read the course description, instructor details, and other information

### 5.2 Enrolling in Courses
1. **Find a Course**:
   - Browse the course catalog or search for specific courses

2. **Enroll in the Course**:
   - Click the "Enroll" button on the course details page
   - You will be enrolled in the course and can start learning immediately

### 5.3 Tracking Course Progress
1. **Access Your Courses**:
   - Go to your profile page
   - Click on the "Courses" tab
   - View your enrolled courses with progress indicators

2. **Update Progress**:
   - As you complete parts of the course, your progress will be automatically updated
   - You can also manually update your progress if needed

### 5.4 Creating Courses
1. **Access the Course Creator**:
   - Click the "Create" button in the sidebar
   - Select "Create Course" from the dropdown menu

2. **Fill in the Course Form**:
   - **Title**: Enter a descriptive title for your course
   - **Description**: Provide a detailed course description
   - **Instructor**: Enter the instructor's name
   - **Duration**: Specify the course duration
   - **Level**: Select the course difficulty level (Beginner, Intermediate, Advanced)
   - **Category**: Select an appropriate course category
   - **Cover Image**: Upload a cover image for your course

3. **Submit Your Course**:
   - Click the "Publish" button
   - Your course will be published and available for enrollment

## 6. Social Features

### 6.1 Viewing User Profiles
1. **Find User Profiles**:
   - Click on a user's name or avatar in content, courses, or messages
   - Search for users using the search functionality

2. **View Profile Details**:
   - User information and bio
   - Created content and courses

### 6.2 Adding Friends
1. **Navigate to a User's Profile**:
   - Click on the user's name or avatar

2. **Send a Friend Request**:
   - Click the "Add Friend" button on their profile
   - The user will receive a friend request notification

3. **Accept Friend Requests**:
   - Click on the notifications icon
   - Find pending friend requests
   - Click "Accept" to confirm the friendship

### 6.3 Sending Messages
1. **Access the Messages Page**:
   - Click on "Messages" in the sidebar

2. **Start a New Conversation**:
   - Click the "New Message" button
   - Select a friend from your contacts
   - Enter your message content
   - Click "Send"

3. **View and Respond to Messages**:
   - Click on a conversation in the messages list
   - Read the message history
   - Enter your response and click "Send"

### 6.4 Interacting with Content
1. **Like Content**:
   - Click the like button on any content post

2. **Comment on Content**:
   - Click the comment button on any content post
   - Enter your comment in the text box
   - Click "Post Comment"

3. **Share Content**:
   - Click the share button on any content post
   - Select a sharing method (direct message, social media, etc.)

1. **Access the Backup Section**:
   - Navigate to the "System Settings" tab in the admin dashboard
   - Scroll to the "Data Backup Settings" section

2. **Create a Backup**:
   - Click the "Manual Backup Now" button
   - The backup file will be downloaded to your device

### 8.2 Restoring from Backup
1. **Access the Restore Section**:
   - Navigate to the "System Settings" tab in the admin dashboard
   - Scroll to the "Data Backup Settings" section

2. **Restore from Backup**:
   - Click the "Restore Backup" button
   - Select the backup file from your device
   - Click "Upload" to start the restoration process
   - You will receive a confirmation message when restoration is complete

## 9. Troubleshooting

### 9.1 Common Issues and Solutions

#### Login Issues
- **Problem**: Invalid email or password
  **Solution**: Double-check your credentials, ensure caps lock is off, or reset your password

- **Problem**: Account locked
  **Solution**: Contact admin for assistance

#### Content Issues
- **Problem**: Cannot upload images
  **Solution**: Check file size and format (JPG, PNG, GIF up to 10MB)

- **Problem**: Content not visible
  **Solution**: Ensure content is published and not in draft status

#### Course Issues
- **Problem**: Cannot enroll in course
  **Solution**: Ensure you are logged in and the course is published

- **Problem**: Progress not updating
  **Solution**: Refresh the page or try clearing your browser cache

#### Social Features Issues
- **Problem**: Cannot send messages
  **Solution**: Ensure you are connected to the internet and the recipient is your friend

- **Problem**: Friend requests not sending
  **Solution**: Check if the user has reached their friend limit or has blocked you

### 9.2 Browser Compatibility
If you experience issues with the platform, try the following:
1. **Clear Browser Cache**:
   - Instructions vary by browser, but generally found in Settings > Privacy

2. **Update Browser**:
   - Ensure you are using the latest version of your browser

3. **Try a Different Browser**:
   - If issues persist, try accessing the platform with a different browser

### 9.3 Contacting Support
If you encounter persistent issues, contact the platform administrator:
- **Email**: admin@knowledge-nebula.com
- **Support Form**: Available in the footer of the platform

## 10. Best Practices

### 10.1 Content Creation Guidelines
- **Quality**: Create original, well-researched content
- **Formatting**: Use headings, bullet points, and images to enhance readability
- **Citations**: Cite sources when referencing external information
- **Categories**: Use appropriate categories for better discoverability
- **Titles**: Use clear, descriptive titles

### 10.2 Course Creation Guidelines
- **Structure**: Organize courses into logical sections
- **Content Quality**: Ensure course materials are accurate and up-to-date
- **Instructor Information**: Provide clear instructor credentials
- **Course Materials**: Include a variety of materials (text, images, videos)
- **Assessment**: Include quizzes or assignments to reinforce learning

### 10.3 Social Interaction Guidelines
- **Respect**: Treat other users with respect and courtesy
- **Constructive Feedback**: Provide helpful, constructive feedback
- **Privacy**: Respect other users' privacy and personal information
- **Relevance**: Keep discussions relevant to the topic at hand
- **Safety**: Report any inappropriate content or behavior to admins

### 10.4 Security Best Practices
- **Password Strength**: Use strong, unique passwords
- **Account Security**: Don't share your account credentials
- **Phishing Awareness**: Be cautious of suspicious emails or links
- **Logout**: Always logout when using public computers
- **Updates**: Keep your browser and security software updated

## 11. Glossary

### 11.1 Platform Terminology

| Term | Definition |
|------|------------|
| Content | Knowledge-based articles, posts, or resources created by users |
| Course | Structured learning materials with defined objectives and duration |
| Enrollment | The process of signing up for a course |
| Progress | The percentage of a course that has been completed |
| Groups | Communities of users with shared interests |
| Messages | Private communications between users |
| Friends | Connections between users that enable direct communication |
| Admin | User with elevated privileges for system management |
| Backup | A copy of system data for disaster recovery |
| Operation Log | Record of system actions for audit and troubleshooting |

### 11.2 Technical Terms

| Term | Definition |
|------|------------|
| JWT | JSON Web Token, used for secure authentication |
| Supabase | The database and backend service used by the platform |
| React | The JavaScript library used for building the user interface |
| TypeScript | A typed superset of JavaScript used for development |
| Tailwind CSS | The CSS framework used for styling the platform |
| API | Application Programming Interface, used for frontend-backend communication |
| Responsive Design | Design that adapts to different screen sizes |
| Lazy Loading | Technique to defer loading of non-critical resources |
